## 🌟 Top-Tier Features

* **🤖 AI-Powered Concierge:** Context-aware travel recommendations powered by **Google Gemini AI**.
* **🏨 Unified Booking Engine:** Seamlessly search and book Hotels, Flights, and Car Rentals in one unified interface.
* **💳 Integrated Digital Wallet:** Multi-currency support (USD, PKR) with real-time balance tracking and automated payment processing.
* **🌦️ Live Weather Insights:** Real-time climate data for your destination to help you pack smarter.
* **🔏 Biometric Security:** Secure your travel data and wallet with Fingerprint/FaceID integration.
* **📂 Smart Library:** Access educational travel guides, tips, and cultural insights on the go.

```bash
src/
├── components/   # Gradient-based UI & Reusable Cards
├── navigation/   # Stack & Bottom Tab architecture
├── context/      # AuthProvider & Wallet State management
├── services/     # Gemini AI & Weather API integrations
└── utils/        # PDF Generation & Data export logic

```
## ✨ Smart Travel Planner App Key Highlights

- **AI-Powered Smart Travel Planner** — Intelligent React Native mobile app that generates personalized travel itineraries using Gemini AI
- **Smart Itinerary Generation** — Creates complete day-by-day travel plans based on destination, budget, duration, preferences, and interests
- **Interactive Maps Integration** — Beautiful Mapbox-powered maps to visualize routes, attractions, and trip highlights
- **Real-Time Recommendations** — Weather forecasts, cost estimates, and location-based suggestions powered by AI
- **Budget Optimization** — Smart expense planner that helps users stay within their travel budget
- **Modern React Native Architecture** — Built with TypeScript, Zustand, and clean, scalable code
- **Smooth Animations & UX** — Delightful user experience with Reanimated for fluid interactions
- **Cross-Platform Excellence** — High-quality, responsive design for both iOS and Android

An intelligent and user-friendly travel companion app that leverages AI to make trip planning smarter, faster, and more enjoyable.
